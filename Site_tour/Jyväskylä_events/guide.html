Käyttöohje

Siili Jyväskylä — Tapahtumat & Kalenteri on sovellus, joka kokoaa yhteen Siilin Jyväskylän toimiston tapahtumat sekä lähialueen kaupunkitapahtumat, joilla voi olla vaikutusta työmatkoihin ja pysäköintiin.

Kalenterinäkymä

Kuukausinäkymä

Oletusnäkymä on kuukausikalenteri. Tapahtumat näkyvät värikkäinä palkkeina kunkin päivän kohdalla. Väri kertoo tapahtuman kategorian.

  • Toimistotapahtumat — vasemmassa reunassa valkoinen raita
  • Kaupunkitapahtumat — vasemmassa reunassa tumma raita
  • Tänään-merkintä — nykyinen päivä korostettu sinisellä reunuksella

Liikennevaroitukset

Jos päivällä on merkittävän liikennevaikutuksen tapahtuma, päivän oikeassa yläkulmassa näkyy punainen pallo 🔴. Tämä tarkoittaa, että Lutakon alueen liikenne ja pysäköinti ovat tavanomaista ruuhkaisempia.

Navigointi

Vaihda kuukautta nuolipainikkeilla kalenterin yläpuolella. Klikkaa mitä tahansa tapahtumaa nähdäksesi lisätiedot. Jos päivässä on enemmän kuin 3 tapahtumaa, klikkaa +N lisää -linkkiä nähdäksesi kaikki.

Suodattimet

Tyyppisuodatin

Valitse näytettävien tapahtumien tyyppi:

  • Kaikki — näyttää sekä toimisto- että kaupunkitapahtumat
  • Toimisto — vain Siilin sisäiset tapahtumat (Site Monthly, AI Club, Tech Talk jne.)
  • Kaupunki — vain Jyväskylän kaupunkitapahtumat (festivaalit, konsertit, messut jne.)

Liikennevaikutussuodatin

Suodata kaupunkitapahtumat niiden liikennevaikutuksen perusteella:

  • 🟢 Vähäinen — pieni tapahtuma, ei merkittävää vaikutusta
  • 🟡 Kohtalainen — kohtalainen määrä kävijöitä, parkkipaikat voivat täyttyä
  • 🔴 Merkittävä — suuri tapahtuma, tiekatkoja ja merkittäviä pysäköintiongelmia

Haku ja kategoriat

Hakukentällä voit etsiä tapahtumia nimellä, kuvauksella, paikalla tai kategorialla. Kategoriapainikkeet suodattavat tapahtumia yksittäisten kategorioiden mukaan — voit valita useita.

Vinkki: Kun suodattimia on käytössä, filtteripalkissa näkyy punainen ✕ Nollaa suodattimet -painike, jolla saat kaikki tapahtumat takaisin näkyviin yhdellä klikkauksella.

Karttanäkymä

Karttanäkymä näyttää Siilin toimiston sijainnin sekä tapahtumapaikat interaktiivisella kartalla.

  • Sininen 🏢-merkki — Siilin toimisto, Lutakonaukio 1
  • Värilliset numeromerkit — tapahtumapaikat, joissa luku kertoo tapahtumien määrän
  • Merkkien väri kertoo pahimman liikennevaikutuksen: vihreä (vähäinen), keltainen (kohtalainen), punainen (merkittävä)
  • Klikkaa merkkiä nähdäksesi kyseisen paikan tapahtumat

Tapahtumalista

Tapahtumat-näkymä listaa kaikki tapahtumat aikajärjestyksessä päivämäärittäin ryhmiteltynä. Jokainen kortti näyttää:

  • Kategorian emojin ja värin
  • Kellonajan, keston ja paikan
  • Liikennevaikutusbadgen (kaupunkitapahtumat)
  • Lyhyen kuvauksen

Klikkaa korttia avataksesi lisätiedot modaalissa.

Vienti

  • Lataa .ics — lataa kaikki suodatetut tapahtumat kalenteritiedostona (Google Calendar, Outlook, Apple Calendar)
  • Lataa .csv — lataa tapahtumat Excel-yhteensopivana CSV-tiedostona

Yksittäisen tapahtuman voi lisätä kalenteriin klikkaamalla tapahtuman tietonäkymässä Lisää kalenteriin -painiketta.

Analytiikka

Analytiikkanäkymässä näet yhteenvedon tapahtumista neljänä kaaviona: kategorioittain, kuukausittain, liikennevaikutuksittain ja tapahtumapaikoittain. Yläreunassa näkyvät kokonaisluvut.

Kielivalinta

Vaihda kieltä otsikkorivin oikeassa reunassa olevista 🇫🇮 FI / 🇬🇧 EN -painikkeista. Kielivalinta tallentuu automaattisesti selaimen muistiin.

User Guide

Siili Jyväskylä — Events & Calendar is an application that brings together Siili's Jyväskylä office events and nearby city events that may affect commuting and parking.

Calendar View

Monthly View

The default view is a monthly calendar. Events appear as coloured pills on each day. The colour indicates the event category.

  • Office events — white stripe on the left edge
  • City events — dark stripe on the left edge
  • Today marker — current day highlighted with a blue border

Traffic Warnings

If a day has an event with significant traffic impact, a red dot 🔴 appears in the top-right corner. This means traffic and parking in the Lutakko area will be heavier than usual.

Navigation

Switch months using the arrow buttons above the calendar. Click any event to see details. If a day has more than 3 events, click the +N more link to see all of them.

Filters

Type Filter

Select which type of events to display:

  • All — shows both office and city events
  • Office — only Siili internal events (Site Monthly, AI Club, Tech Talk, etc.)
  • City — only Jyväskylä city events (festivals, concerts, fairs, etc.)

Traffic Impact Filter

Filter city events by their traffic impact level:

  • 🟢 Low — small event, no significant impact
  • 🟡 Medium — moderate number of visitors, parking may fill up
  • 🔴 Significant — large event, road closures and major parking issues

Search and Categories

Use the search field to find events by name, description, location, or category. Category chips filter events by individual categories — you can select multiple.

Tip: When filters are active, a red ✕ Reset filters button appears in the filter bar, allowing you to restore all events with a single click.

Map View

The map view shows the Siili office location and event venues on an interactive map.

  • Blue 🏢 marker — Siili office at Lutakonaukio 1
  • Coloured number markers — event venues, where the number indicates the event count
  • Marker colour indicates the worst traffic impact: green (low), yellow (medium), red (significant)
  • Click a marker to see events at that venue

Events List

The Events view lists all events chronologically, grouped by date. Each card shows:

  • Category emoji and colour
  • Time, duration, and location
  • Traffic impact badge (city events)
  • Short description

Click a card to open the full details in a modal.

Export

  • Download .ics — download all filtered events as a calendar file (Google Calendar, Outlook, Apple Calendar)
  • Download .csv — download events as an Excel-compatible CSV file

You can also add a single event to your calendar by clicking Add to calendar in the event detail view.

Analytics

The Analytics view provides a summary of events in four charts: by category, by month, by traffic impact, and by venue. Summary totals are shown at the top.

Language

Switch language using the 🇫🇮 FI / 🇬🇧 EN buttons in the top-right corner of the header. Your selection is saved automatically in the browser.
