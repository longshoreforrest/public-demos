Käyttöohje

Asiakasreferenssit on interaktiivinen sovellus Siili Solutionsin asiakasprojektien ja referenssien selaamiseen, suodattamiseen ja analysointiin.

Referenssit-näkymä

Selaaminen

Päänäkymässä näet kaikki asiakasreferenssit kortteina. Jokainen kortti näyttää asiakkaan nimen, projektin otsikon, tiivistelmän, teknologiat ja tulokset.

Suodattaminen

  • Kategoria: Valitse kategoriapainikkeista haluamasi osa-alue (AI, Data, Cloud, Kehitys, Transformaatio, IoT, Tietoturva, Design).
  • Toimiala: Suodata referenssejä toimialan mukaan.
  • Haku: Kirjoita hakukenttään etsiäksesi asiakkaan nimellä, teknologialla tai toimialalla.

Referenssin tiedot

Klikkaa referenssikorttia avataksesi yksityiskohtaisen näkymän. Näet haasteen, ratkaisun, lopputuloksen ja asiakkaan palautteen.

Yhteenveto-näkymä

Dashboard näyttää yhteenvedon kaikista referensseistä visuaalisina kaavioina:

  • Referenssit toimialoittain: Pylväskaavio toimialajakaumasta.
  • Referenssit kategorioittain: Kaavio osa-aluejakaumasta.
  • Käytetyimmät teknologiat: Yleisimpien teknologioiden yhteenveto.

Kielivalinta

Vaihda kieltä oikean yläkulman FI/EN-painikkeilla. Kielivalinta tallentuu automaattisesti selaimen muistiin.

User Guide

Customer References is an interactive application for browsing, filtering and analyzing Siili Solutions' client projects and references.

References View

Browsing

The main view displays all customer references as cards. Each card shows the client name, project title, summary, technologies and results.

Filtering

  • Category: Select category chips to filter by area (AI, Data, Cloud, Development, Transformation, IoT, Security, Design).
  • Industry: Filter references by industry.
  • Search: Type in the search field to find references by client name, technology or industry.

Reference Details

Click a reference card to open a detailed view. You'll see the challenge, solution, outcome and customer feedback.

Summary View

The dashboard shows a visual summary of all references:

  • References by industry: Bar chart of industry distribution.
  • References by category: Chart of area distribution.
  • Top technologies: Overview of most used technologies.

Language

Switch language using the FI/EN buttons in the top-right corner. Your selection is saved automatically.
