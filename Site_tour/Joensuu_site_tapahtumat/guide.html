Käyttöohje

Joensuu Site Tapahtumat on interaktiivinen sovellus Siili Solutionsin Joensuun toimiston tapahtumien hallintaan ja selaamiseen.

Tapahtumat-näkymä

Selaaminen

Päänäkymässä näet kaikki tulevat tapahtumat aikajärjestyksessä. Jokainen tapahtuma näyttää päivämäärän, kellonajan, paikan ja kuvauksen.

Suodattaminen

  • Kategoria: Klikkaa kategoriapainikkeita valitaksesi yhden tai useamman kategorian. Voit valita useita samanaikaisesti.
  • Päivämäärä: Aseta aikaväli "Alkaen"- ja "Saakka"-kentillä.
  • Haku: Kirjoita hakukenttään etsiäksesi tapahtumia nimellä, kuvauksella tai paikalla.

Kalenteriin lisääminen

Klikkaa minkä tahansa tapahtuman kohdalla Lisää kalenteriin -painiketta ladataksesi .ics-tiedoston. Tiedoston voi avata Google Calendarissa, Outlookissa tai Apple Calendarissa.

Kalenterinäkymä

Kuukausinäkymässä näet tapahtumat visuaalisesti kalenterissa. Klikkaa tapahtumaa nähdäksesi lisätiedot. Vaihda kuukautta nuolipainikkeilla.

Slack-integraatio

Simuloitu Slack-näkymä näyttää miltä tapahtumailmoitukset näyttäisivät Slack-kanavilla. Voit:

  • Selata eri kanavia (tapahtumat, AI-klubi, liikunta, lautapelit)
  • Lähettää viestejä ja saada bot-vastauksen
  • Simuloida ilmoituspopupin "Simuloi ilmoitus" -painikkeella
  • Säätää ilmoitusasetuksia

Vienti

Excel (.csv)

Lataa kaikki tapahtumat CSV-tiedostona, jonka voi avata Excelissä. Tiedosto sisältää kaikki tiedot puolipisteellä erotettuna ja tukee skandinaavisia merkkejä.

Kalenteri (.ics)

Lataa kaikki tapahtumat yhtenä .ics-kalenteritiedostona. Voit tuoda sen mihin tahansa kalenterisovellukseen.

Molemmissa vientivaihtoehdoissa voit valita viedäänkö kaikki vai vain suodatetut tapahtumat.

Kielivalinta

Vaihda kieltä oikeasta yläkulmasta FI/EN-painikkeilla. Kielivalinta tallentuu automaattisesti.

User Guide

Joensuu Site Events is an interactive application for browsing and managing events at Siili Solutions' Joensuu office.

Events View

Browsing

The main view shows all upcoming events in chronological order. Each event displays the date, time, location and description.

Filtering

  • Category: Click category chips to select one or more categories. Multiple selections are supported.
  • Date range: Set a date range using the "From" and "To" fields.
  • Search: Type in the search field to find events by name, description or location.

Adding to Calendar

Click the Add to calendar button on any event to download an .ics file. This file can be opened in Google Calendar, Outlook, or Apple Calendar.

Calendar View

The monthly view shows events visually on a calendar grid. Click any event to see details. Navigate between months using the arrow buttons.

Slack Integration

The simulated Slack view shows how event notifications would look on Slack channels. You can:

  • Browse different channels (events, AI club, sports, board games)
  • Send messages and receive bot replies
  • Simulate a notification popup with the "Simulate notification" button
  • Configure notification settings

Export

Excel (.csv)

Download all events as a CSV file that can be opened in Excel. The file uses semicolons as separators and supports Scandinavian characters.

Calendar (.ics)

Download all events as a single .ics calendar file. You can import this into any calendar application.

Both export options allow you to export all events or only the currently filtered ones.

Language

Switch language using the FI/EN buttons in the top-right corner. Your selection is saved automatically.
