Käyttöohje

Siili Solutions - Asiakasreferenssien hallinta

Yleiskuvaus

Tämä sovellus on Siili Solutionsin asiakasreferenssien hallintaportaali. Sovelluksen avulla voit selata, suodattaa ja analysoida Siilin asiakasprojekteja, käytettyjä teknologioita, asiantuntijaprofiileja ja kilpailutuksiin liittyviä referenssejä.

Näkymät

Referenssit

Päänakyma, jossa näet kaikki asiakasreferenssit listamuodossa. Voit suodattaa toimialan mukaan tai hakea vapaatekstillä. Klikkaa referenssiä nähdäksesi lisätiedot.

Värikoodatut tagit:

  • Toimiala - Projektin toimiala
  • Teknologia - Käytetty teknologia
  • Tyyppi - Avainsana
  • Kilpailutus - Kilpailutusreferenssi

Asiakkaat

Asiakaskohtainen näkymä, jossa näet jokaisen asiakkaan projektit koottuna yhteen. Klikkaa asiakaskorttia nähdäksesi kootut tiedot.

Teknologiat

Teknologiakohtainen näkymä, jossa näet kaikki projekteissa käytetyt teknologiat kategorioittain. Jokaisen teknologian vieressä näkyy kuinka monessa projektissa sitä on käytetty. Klikkaa teknologiaa nähdäksesi projektit.

Asiantuntijat

Siilin asiantuntijoiden profiilit sisältäen osaamiskuvaukset, toimialakokemuksen, sertifikaatit ja projektit joissa asiantuntija on ollut mukana.

Kilpailutus

Kilpailutusreferenssit vaatimuslomakemuodossa. Tämä näkymä on suunniteltu erityisesti julkisten hankintojen referenssikuvauksiin, kuten Ruokaviraston osa-alueen 1 (sovelluskehitys ja ylläpito) vaatimuksiin.

Datan käsittely

Lähtödata on johdettu konsulttien CV-tiedoista ja kilpailutusreferenssidokumenteista. Henkilöiden nimet on anonymisoitu. Asiantuntijaprofiilien tiedot on koostettu useista lähteistä yhtenäiseksi referenssikuvaukseksi.

Asiakkuusreferenssit on muodostettu yhdistämällä samassa asiakkuudessa olleiden henkilöiden profiilit ja projektikuvaukset.

Kielen vaihto

Sovellus tukee suomea ja englantia. Kieli vaihdetaan oikean yläkulman kielivalitsimella (FI / EN). Valittu kieli tallennetaan selaimen muistiin.

Tekninen toteutus

Sovellus on toteutettu puhtaalla HTML, CSS ja JavaScriptillä ilman ulkoisia kirjastoja. Kaikki data on staattisessa data.js-tiedostossa.

User Guide

Siili Solutions - Customer Reference Management

Overview

This application is Siili Solutions' customer reference management portal. It allows you to browse, filter and analyze Siili's client projects, technologies used, expert profiles and tender-related references.

Views

References

The main view where you can see all customer references in a list format. You can filter by industry or search with free text. Click a reference to see more details.

Color-coded tags:

  • Industry - Project industry
  • Technology - Technology used
  • Type - Keyword
  • Tender - Tender reference

Clients

Client-specific view where you can see all projects grouped by client. Click a client card to view the compiled details.

Technologies

Technology-specific view showing all technologies used across projects, organized by category. Each technology displays the number of projects where it has been used. Click a technology to see the related projects.

Experts

Siili expert profiles including competence descriptions, industry experience, certifications and projects the expert has been involved in.

Tender

Tender references in a requirements form format. This view is specifically designed for public procurement reference descriptions, such as the Finnish Food Authority's area 1 (application development and maintenance) requirements.

Data Processing

The source data is derived from consultant CV data and tender reference documents. Personal names have been anonymized. Expert profile information has been compiled from multiple sources into a unified reference description.

Client references have been formed by combining the profiles and project descriptions of individuals who have worked within the same client engagement.

Language Selection

The application supports Finnish and English. Switch the language using the language selector (FI / EN) in the top-right corner. The selected language is saved in the browser's local storage.

Technical Implementation

The application is built with plain HTML, CSS and JavaScript without external libraries. All data is stored in a static data.js file.
